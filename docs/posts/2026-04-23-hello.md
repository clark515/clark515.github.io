# 开站声明

在知乎、freebuf、安全客发文各有局限——平台审核、排版受限、数据归属不在我。这里是技术思考的沉淀地。

## 写什么

围绕四个方向：

- **传统安全**：Web / Java / 反序列化、二进制、渗透
- **安全运营 / 检测**：SOC、入侵检测、事件响应
- **安全体系**：攻击链建模、场景驱动方法论、能力评估
- **AI 安全（重点方向）**：Agent / MCP、大模型红队、AI 辅助安全

## 原则

- 讲真问题，不炒概念
- 尽量给可复现的 PoC / 代码 / 实验环境
- 做不到"总结行业"，只做"梳理自己思考"

## 技术栈

站点基于 [VitePress](https://vitepress.dev/) 搭建，markdown 写作、GitHub Actions 自动部署。源码在 [clark515/clark515.github.io](https://github.com/clark515/clark515.github.io)。
